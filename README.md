# GenAI Chatbot (Beginner Project)

## Description
A simple web-based chatbot built using Flask and Hugging Face transformer models.

## Tech Stack
- Python
- Flask
- Hugging Face Transformers
- HTML/CSS/JS

## Features
- Basic conversational chatbot
- Web interface
- Free AI model integration

## Limitations
- Uses lightweight open-source model (not as advanced as GPT-4)
- Responses may vary in quality

## Learning Outcome
- Learned how AI chatbots work
- Understood API and model integration
- Built full-stack web application
